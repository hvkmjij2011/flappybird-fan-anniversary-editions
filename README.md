# Flappy Bird: 13th Anniversary Revival

[TR] Bu proje, orijinal 2013 Flappy Bird deneyimini modern bir altyapı ile birleştiren, hiçbir kâr amacı gütmeyen bir topluluk projesidir.

[EN] This project is a non-profit community effort that revives the 2013 Flappy Bird experience with modern infrastructure.


## 🛠 Üretim Süreci / Production Process

[TR] Bu çalışma, orijinal APK'nın Smali seviyesinde tersine mühendislik ile parçalara ayrılıp, Supabase entegrasyonu ile yeniden birleştirilmesiyle ortaya çıktı. Orijinal boyutu korumak için fizik motoruna dokunulmadı.

[EN] This project was built by reverse-engineering the original APK at the Smali level and integrating a Supabase backend. The physics engine remains untouched to preserve the original footprint.


## 📊 Skor Tablosu / Leaderboard Mechanics

[TR] Skor verileri, her "new best" anında internet bağlantınız aktifse otomatik olarak Supabase veritabanına iletilir. Veriler, kalıcı ücretsiz bir katmanda üç ana parametre ile saklanır:

 1- User ID: Android cihaz kimliğinizin (Android-ID) güvenli bir şekilde hashlenmesiyle oluşturulur; böylece oyunu silip tekrar yükleseniz dahi skorlarınız kimliğinizle eşleşmeye devam eder.

2- Display Name: İlk skor gönderiminizde belirlenir ve skor tablosunda görünecek olan isminizdir.

3- Score: Ulaştığınız en yüksek puan.

[EN] Score data is automatically transmitted to our persistent Supabase database on every "new best" event, provided you are connected to the internet. Data is stored using three parameters:

1- User ID: Generated by hashing your Android-ID. This ensures that even if you uninstall and reinstall the game, your identity remains consistent on the leaderboard.

2- Display Name: Defined during your first submission, representing your public identity on the leaderboard.

3- Score: Your achieved high score.
